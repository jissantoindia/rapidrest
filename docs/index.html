CLI Tools

RapidRest comes with powerful CLI tools to help you manage your application:

Available Commands

# Create a new migration
./rapid make:migration create_users_table

# Run all pending migrations
./rapid migrate

# Rollback the last batch of migrations
./rapid migrate:rollback

# Refresh the entire database
./rapid migrate:refresh

# Check migration status
./rapid migrate:status
